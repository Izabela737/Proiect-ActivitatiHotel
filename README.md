# Proiect-ActivitatiHotel

Aplicație web pentru gestionarea activităților unui hotel. Permite rezervări de camere, cereri de curățenie și administrarea utilizatorilor (client, angajat, manager).

## Funcționalități principale

- **Client**: vizualizare camere, cerere rezervare, cerere curățenie, vizualizare istoricul rezervărilor.  
- **Employee**: vizualizare camere pentru curățenie, actualizare status curățenie.  
- **Manager**: aprobare/refuz rezervări, gestionare camere, alocare angajați.
